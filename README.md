# 🏡 Edmonton Property Pricing Analysis

This capstone project analyzes over 300,000 residential transactions in Edmonton to determine how property features (like bathrooms, garages, and crime rates) impact home prices. 

## 📌 Key Highlights
- Used Random Forest and XGBoost to predict pricing (R² = 0.92)
- Feature importance & SHAP analysis to interpret impact
- Integrated property, crime, and geographic metadata
- Visualized insights via Tableau dashboard

## 📁 Included Files
- Jupyter Notebook (Modeling)
- Report (PDF)
- Final Slide Deck (PPTX)

## 🔧 Tools Used
Python, Pandas, Scikit-Learn, XGBoost, Tableau, SHAP

## Real-World Applications
- Helps buyers/investors identify value-adding features
- Guides agents in pricing strategy
- Supports urban planning with spatial insights
